Administration av material
==========================

Användare kan hantera en grundläggande databas av material, organiserat efter kategorier.

Kategorier är behållare till vilka specifika material och andra kategorier kan tilldelas. De lagras i en hierarkisk trädstruktur, eftersom material kan tillhöra bladkategorier eller mellanliggande kategorier.

För att hantera kategorier måste användare följa dessa steg:

*   Gå till alternativet "Administration -> Material".
*   Programmet visar ett träd med kategorier.
*   Ange ett kategorinamn i textrutan och klicka sedan på "Lägg till."
*   Programmet lägger till kategorin i trädet.

För att infoga en kategori i kategoriträdet måste användare först välja den överordnade kategorin i trädet och sedan klicka på "Lägg till."

.. figure:: images/material.png
   :scale: 50

   Skärm för materialadministration

För att hantera material måste användare följa dessa steg:

*   Välj den kategori som material ska läggas till i och klicka på "Lägg till" till höger om "Material."
*   Programmet lägger till en ny tom rad med fält för att ange detaljer om materialet:

    *   **Kod:** Materialtypskod (detta kan vara den externa koden från ett ERP-system).
    *   **Beskrivning:** Beskrivning av materialet.
    *   **Enhetspris:** Enhetspris för varje materialenhet.
    *   **Enhet:** Enhet som används för att mäta varje materialenhet.
    *   **Kategori:** Kategorin som materialet tillhör.
    *   **Tillgänglighet:** Anger om materialet är aktivt eller inte.

*   Användare fyller i fälten och klickar på "Spara."

Tilldelning av material till projektelement förklaras i kapitlet om "Projekt."
