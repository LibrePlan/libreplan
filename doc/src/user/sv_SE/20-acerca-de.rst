Offentlig finansiering
======================

Det här projektet tar för närvarande emot donationer via organisationen Open Collective <https://opencollective.com/LibrePlan>.

Inom det globala tillämpningsområde som LibrePlan är utformat för när det gäller planeringshantering, utvecklades ett projekt för att lösa några vanliga planeringsproblem.

Det här projektet finansierades delvis av Xunta de Galicia, Ministerio de Industria, Turismo e Comercio och av Europeiska unionen, Fondo Europeo de Desenvolvemento Rexional.
