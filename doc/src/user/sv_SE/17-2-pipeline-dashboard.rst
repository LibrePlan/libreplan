Pipeline-instrumentpanel
########################

.. contents::

Pipeline-instrumentpanelen är ett *LibrePlan*-perspektiv som listar alla företagets projekt i en tabell. Den här "pipeline"-tabellen har nio kolumner, var och en motsvarar en projektstatus:

   *   PRE-SALES
   *   OFFERED
   *   OUTSOURCED
   *   ACCEPTED
   *   STARTED
   *   ON HOLD
   *   FINISHED
   *   CANCELLED
   *   ARCHIVED

Projekt listas med namn i varje kolumn baserat på deras aktuella status.

Ett projekts startdatum, slutdatum och aktuella framsteg kan visas i ett verktygstips genom att hålla muspekaren över projektets namn.

Projekt med statusen *ARCHIVED* kan döljas genom att avmarkera kryssrutan "Visa arkiverad kolumndata".
