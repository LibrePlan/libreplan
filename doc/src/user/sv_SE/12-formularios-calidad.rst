Administration av kvalitetsformulär
=====================================

Kvalitetsformulär består av en lista med frågor eller påståenden som anger de uppgifter eller processer som bör ha slutförts innan en uppgift kan markeras som avslutad av företaget. Dessa formulär innehåller följande fält:

*   **Namn:** Kvalitetsformulärets namn.
*   **Beskrivning:** En beskrivning av kvalitetsformuläret.
*   **Typ av kvalitetsformulär:** Typen kan ha två värden:

    *   **Procent:** Anger att frågorna måste besvaras i logisk ordning och att jakande svar anger att uppgiften fortskrider. Till exempel kan en grundläggande process för en uppgift antyda att uppgiften är 15% klar. Användare måste svara på en fråga innan de fortsätter till nästa.
    *   **Element:** Anger att frågorna inte behöver besvaras i logisk ordning och kan besvaras i valfri sekvens.

Användare måste följa dessa steg för att hantera kvalitetsformulär:

*   Från menyn "Administration," öppna alternativet "Kvalitetsformulär."
*   Klicka på "Redigera" för ett befintligt formulär eller klicka på "Skapa" för att skapa ett nytt.
*   Programmet visar ett formulär med fält för namn, beskrivning och typ.
*   Välj typen.
*   Programmet visar de fält som är tillåtna för varje typ:

    *   **Procent:** Fråga och procent.
    *   **Element:** Fråga.

*   Klicka på "Spara" eller "Spara och fortsätt."
