LibrePlan: Användardokumentation

Detta dokument innehåller nödvändig hjälp för att använda LibrePlan, den öppen källkodsbaserade webbapplikationen för projektplanering.

Dokumentet är indelat i tre huvudavsnitt:

För det första, globala mål och allmänt beteende.

För det andra, grundläggande entiteter för att förstå de minimala koncept som krävs för att använda LibrePlan.

Slutligen, en fullständig processbeskrivning för att skapa beställningar, projekt, projektplanering, resurstilldelning, framstegsregistrering och resultatuttag.

Vänligen stöd LibrePlan

LibrePlan kan inte existera utan ditt stöd. Vänligen läs donationssidan och överväg att hjälpa oss att fortsätta utvecklingen.
