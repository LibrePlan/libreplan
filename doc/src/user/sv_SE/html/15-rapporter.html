Rapporter

LibrePlan integreras med JasperReports för rapporthantering, vilket gör det möjligt för användare att generera olika rapporter för att analysera data som finns tillgängliga i programmet.

Tillgängliga rapporter:

  • Rapport: Arbetade timmar per resurs
  • Rapport: Totalt antal timmar per resurs och månad
  • Rapport: Arbete och framsteg per projekt
