Данный документ содержит необходимую справку по использованию LibrePlan — веб-приложения с открытым исходным кодом для планирования проектов.

Документ разделён на три основных раздела:

Во-первых, общие цели и поведение приложения.

Во-вторых, основные сущности для понимания минимальных концепций, необходимых для работы с LibrePlan.

Наконец, полное описание процессов создания заказов, проектов, планирования проектов, назначения ресурсов, учёта прогресса и извлечения результатов.
