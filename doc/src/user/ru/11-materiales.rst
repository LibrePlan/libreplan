Администрирование материалов
============================

Пользователи могут управлять базовой базой данных материалов, организованных по категориям.

Категории — это контейнеры, которым можно назначать конкретные материалы и другие категории. Они хранятся в иерархической древовидной структуре, так как материалы могут принадлежать листовым категориям или промежуточным категориям.

Для управления категориями пользователи должны выполнить следующие шаги:

*   Перейти к пункту «Администрирование -> Материалы».
*   Программа отображает дерево категорий.
*   Ввести название категории в текстовое поле и нажать «Добавить».
*   Программа добавляет категорию в дерево.

Чтобы вставить категорию в дерево категорий, пользователи должны сначала выбрать родительскую категорию в дереве, а затем нажать «Добавить».

.. figure:: images/material.png
   :scale: 50

   Экран администрирования материалов

Для управления материалами пользователи должны выполнить следующие шаги:

*   Выбрать категорию, в которую нужно добавить материалы, и нажать «Добавить» справа от «Материалы».
*   Программа добавляет новую пустую строку с полями для ввода сведений о материале:

    *   **Код:** Код типа материала (может быть внешним кодом из ERP).
    *   **Описание:** Описание материала.
    *   **Цена за единицу:** Цена за единицу каждого материала.
    *   **Единица:** Единица, используемая для измерения каждой единицы материала.
    *   **Категория:** Категория, к которой принадлежит материал.
    *   **Доступность:** Указывает, активен ли материал.

*   Пользователи заполняют поля и нажимают «Сохранить».

Назначение материалов элементам проекта описано в главе о «Проектах».
