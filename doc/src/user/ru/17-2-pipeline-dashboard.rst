Дашборд конвейера проектов
##########################

.. contents::

Дашборд конвейера проектов — это перспектива *LibrePlan*, в которой все проекты компании отображаются в виде таблицы. Эта таблица «конвейера» содержит девять столбцов, каждый из которых соответствует определённому статусу проекта:

   *   PRE-SALES (Предпродажа)
   *   OFFERED (Предложен)
   *   OUTSOURCED (Аутсорсинг)
   *   ACCEPTED (Принят)
   *   STARTED (Начат)
   *   ON HOLD (Приостановлен)
   *   FINISHED (Завершён)
   *   CANCELLED (Отменён)
   *   ARCHIVED (В архиве)

Проекты перечислены по имени в каждом столбце в соответствии с их текущим статусом.

Дату начала, дату окончания и текущий прогресс проекта можно просмотреть во всплывающей подсказке, наведя курсор мыши на название проекта.

Проекты со статусом *ARCHIVED* можно скрыть, сняв флажок «Показывать данные архивного столбца».
