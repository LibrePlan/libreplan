Управление пользователями
=========================

Система LibrePlan позволяет администраторам управлять профилями пользователей, авторизациями и самими пользователями. Пользователи назначаются профилям, которые могут иметь ряд предопределённых ролей, предоставляющих доступ к функциям программы. Роли — это предопределённые авторизации в LibrePlan. Примеры ролей:

*   **Администрирование:** Роль, которая должна быть назначена администраторам для выполнения административных операций.
*   **Читатель веб-сервисов:** Роль, необходимая пользователям для просмотра данных веб-сервисов программы.
*   **Писатель веб-сервисов:** Роль, необходимая пользователям для записи данных через веб-сервисы программы.

Роли предопределены в системе. Профиль пользователя состоит из одной или нескольких ролей. Пользователи должны иметь определённые роли для выполнения тех или иных операций.

Пользователям можно назначить один или несколько профилей либо одну или несколько ролей непосредственно, что позволяет предоставлять специфические или общие авторизации.

Для управления пользователями выполните следующие шаги:

*   Перейдите в «Управление пользователями» в меню «Администрирование».
*   Программа отображает форму со списком пользователей.
*   Нажмите кнопку редактирования нужного пользователя или нажмите кнопку «Создать».
*   Появится форма со следующими полями:

    *   **Имя пользователя:** Имя для входа пользователя.
    *   **Пароль:** Пароль пользователя.
    *   **Авторизован/Не авторизован:** Параметр для включения или отключения учётной записи пользователя.
    *   **Email:** Адрес электронной почты пользователя.
    *   **Список связанных ролей:** Для добавления новой роли необходимо найти роль в списке выбора и нажать «Назначить».
    *   **Список связанных профилей:** Для добавления нового профиля необходимо найти профиль в списке выбора и нажать «Назначить».

Управление профилями
--------------------

Для управления профилями программы необходимо выполнить следующие шаги:

*   Перейдите в «Управление профилями пользователей» в меню «Администрирование».
*   Программа отображает список профилей.
*   Нажмите кнопку редактирования нужного профиля или нажмите «Создать».
*   В программе появится форма со следующими полями:

    *   **Название:** Название профиля пользователя.
    *   **Список ролей (авторизаций):** Для добавления роли к профилю необходимо выбрать роль из списка ролей и нажать «Добавить».

.. figure:: images/manage-user-profile.png
   :scale: 50

   Управление профилями пользователей

*   Нажмите «Сохранить» или «Сохранить и продолжить», и система сохранит созданный или изменённый профиль.
