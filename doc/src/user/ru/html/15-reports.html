Отчёты

LibrePlan интегрируется с JasperReports для управления отчётами, позволяя пользователям формировать различные отчёты для анализа данных, доступных в программе.

Доступные отчёты:

  • Отчёт об отработанных часах по ресурсу
  • Отчёт об общем количестве часов, отработанных ресурсом за месяц
  • Отчёт о работе и прогрессе по проекту
