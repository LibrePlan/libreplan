open-source

LibrePlan — это бесплатное веб-приложение с открытым исходным кодом для планирования проектов. Исходный код находится в открытом доступе, и любой желающий может использовать, изучать, изменять и распространять приложение. Лицензионный сбор отсутствует.

Разработка и поддержка проекта такого масштаба требует постоянных усилий со стороны его участников. Поскольку LibrePlan не взимает плату за своё программное обеспечение, проект зависит от добровольных взносов — как в виде кода, так и в виде финансовых пожертвований — для продолжения работы.

Кнопка пожертвования на странице входа

Открывая LibrePlan, вы увидите кнопку Пожертвовать на экране входа. Эта кнопка — прямое приглашение поддержать проект финансово. Пожертвования помогают финансировать:

  • Текущее обслуживание и исправление ошибок
  • Новые функции и улучшения
  • Расходы на инфраструктуру (серверы, CI, хостинг)
  • Работу по документированию и переводу

Даже небольшой разовый взнос имеет реальное значение для добровольцев и разработчиков, которые поддерживают работу приложения. Если LibrePlan полезен вам или вашей организации, пожалуйста, рассмотрите возможность нажать кнопку и сделать пожертвование.

Версия по подписке для предприятий

Версия LibrePlan по подписке (размещённая или локальная) доступна для предприятий через нашу дочернюю компанию LibrePlan Enterprise для тех, кто предпочитает управляемый сервис с гарантированной поддержкой и соглашениями об уровне обслуживания. Подписчики этой версии не увидят кнопку пожертвования на экране входа, поскольку они уже вносят вклад в устойчивость проекта через свою абонентскую плату.

Если вы используете общедоступную (самостоятельно размещённую) версию, кнопка пожертвования будет отображаться как напоминание о том, что проект зависит от щедрости своих пользователей.
