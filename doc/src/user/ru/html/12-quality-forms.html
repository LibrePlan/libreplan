Администрирование форм качества

Формы качества состоят из списка вопросов или утверждений, указывающих задачи или процессы, которые должны быть выполнены, прежде чем задача может быть отмечена как завершённая компанией. Эти формы включают следующие поля:

  • Название: Название формы качества.
  • Описание: Описание формы качества.
  • Тип формы качества: Тип может иметь два значения:
    • Процент: Указывает, что на вопросы необходимо отвечать в логическом порядке, а утвердительные ответы указывают на прогресс задачи. Например, базовый процесс для задачи может предполагать, что задача выполнена на 15%. Пользователи должны ответить на вопрос, прежде чем перейти к следующему.
    • Элемент: Указывает, что на вопросы не обязательно отвечать в логическом порядке, и на них можно отвечать в любой последовательности.

Пользователи должны выполнить следующие шаги для управления формами качества:

  • В меню «Администрирование» перейдите к пункту «Формы качества».
  • Нажмите «Редактировать» для существующей формы или «Создать» для новой.
  • Программа отображает форму с полями для названия, описания и типа.
  • Выберите тип.
  • Программа отображает поля, разрешённые для каждого типа:
    • Процент: Вопрос и процент.
    • Элемент: Вопрос.
  • Нажмите «Сохранить» или «Сохранить и продолжить».
