Назначение

Данный отчёт предоставляет общее количество часов, отработанных каждым ресурсом за указанный месяц. Эта информация может быть полезна для определения сверхурочной работы или, в зависимости от организации, количества часов, за которые каждый ресурс должен получить компенсацию.

Приложение отслеживает рабочие отчёты как для работников, так и для машин. Для машин в отчёте суммируется количество часов их работы в течение месяца.

Входные параметры и фильтры

Для формирования данного отчёта пользователи должны указать год и месяц, за которые они хотят получить общее количество часов, отработанных каждым ресурсом.

Заголовок

В заголовке отчёта отображается:

  • Год, к которому относятся данные отчёта.
  • Месяц, к которому относятся данные отчёта.

Нижний колонтитул

В нижнем колонтитуле отображается дата формирования отчёта.

Тело

Раздел данных отчёта состоит из одной таблицы с двумя столбцами:

  • Один столбец с заголовком Название для имени ресурса.
  • Один столбец с заголовком Часы с общим количеством часов, отработанных ресурсом в данной строке.

Последняя строка содержит суммарное количество часов, отработанных всеми ресурсами за указанный месяц и год.
