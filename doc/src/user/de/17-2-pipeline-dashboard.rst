Pipeline-Dashboard
##################

.. contents::

Das Pipeline-Dashboard ist eine *LibrePlan*-Perspektive, die alle Unternehmensprojekte in einer Tabelle auflistet. Diese „Pipeline"-Tabelle hat neun Spalten, die jeweils einem Projektstatus entsprechen:

   *   PRE-SALES
   *   OFFERED
   *   OUTSOURCED
   *   ACCEPTED
   *   STARTED
   *   ON HOLD
   *   FINISHED
   *   CANCELLED
   *   ARCHIVED

Projekte werden in jeder Spalte basierend auf ihrem aktuellen Status namentlich aufgelistet.

Das Startdatum, Enddatum und der aktuelle Fortschritt eines Projekts können in einem Tooltip-Popup angezeigt werden, indem Sie den Mauszeiger über den Projektnamen bewegen.

Projekte mit dem Status *ARCHIVED* können ausgeblendet werden, indem das Kontrollkästchen „Archivierte Spaltendaten anzeigen" deaktiviert wird.
