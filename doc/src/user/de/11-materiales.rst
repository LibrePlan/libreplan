Verwaltung von Materialien
==========================

Benutzer können eine grundlegende Datenbank von Materialien verwalten, die nach Kategorien organisiert ist.

Kategorien sind Container, denen bestimmte Materialien und andere Kategorien zugewiesen werden können. Sie werden in einer hierarchischen Baumstruktur gespeichert, da Materialien zu Blattkategorien oder Zwischenkategorien gehören können.

Um Kategorien zu verwalten, müssen Benutzer folgende Schritte ausführen:

*   Gehen Sie zur Option „Verwaltung -> Materialien".
*   Das Programm zeigt einen Kategorienbaum an.
*   Geben Sie einen Kategorienamen in das Textfeld ein und klicken Sie dann auf „Hinzufügen".
*   Das Programm fügt die Kategorie zum Baum hinzu.

Um eine Kategorie in den Kategorienbaum einzufügen, müssen Benutzer zuerst die übergeordnete Kategorie im Baum auswählen und dann auf „Hinzufügen" klicken.

.. figure:: images/material.png
   :scale: 50

   Bildschirm zur Materialverwaltung

Um Materialien zu verwalten, müssen Benutzer folgende Schritte ausführen:

*   Wählen Sie die Kategorie aus, zu der Materialien hinzugefügt werden sollen, und klicken Sie rechts neben „Materialien" auf „Hinzufügen".
*   Das Programm fügt eine neue leere Zeile mit Feldern zur Eingabe von Materialdetails hinzu:

    *   **Code:** Materialtypcode (dies kann der externe Code aus einem ERP sein).
    *   **Beschreibung:** Beschreibung des Materials.
    *   **Einheitspreis:** Stückpreis jedes Materialteils.
    *   **Einheit:** Einheit, die zur Messung jeder Materialeinheit verwendet wird.
    *   **Kategorie:** Kategorie, zu der das Material gehört.
    *   **Verfügbarkeit:** Gibt an, ob das Material aktiv ist oder nicht.

*   Benutzer füllen die Felder aus und klicken auf „Speichern".

Die Zuweisung von Materialien zu Projektelementen wird im Kapitel über „Projekte" erläutert.
