Benutzer verwalten

Das System von LibrePlan ermöglicht Administratoren die Verwaltung von Benutzerprofilen, Berechtigungen und Benutzern. Benutzer werden Benutzerprofilen zugewiesen, die eine Reihe vordefinierter Rollen haben können, die den Zugriff auf die Programmfunktionen gewähren. Rollen sind vordefinierte Berechtigungen innerhalb von LibrePlan. Beispiele für Rollen sind:

  • Verwaltung: Eine Rolle, die Administratoren zugewiesen werden muss, damit sie administrative Vorgänge durchführen können.
  • Webservice-Leser: Eine Rolle, die Benutzer benötigen, um die Webservices des Programms abzufragen.
  • Webservice-Schreiber: Eine Rolle, die Benutzer benötigen, um Daten über die Webservices des Programms zu schreiben.

Rollen sind im System vordefiniert. Ein Benutzerprofil besteht aus einer oder mehreren Rollen. Benutzer müssen bestimmte Rollen haben, um bestimmte Vorgänge durchführen zu können.

Benutzern können ein oder mehrere Profile oder direkt eine oder mehrere Rollen zugewiesen werden, wodurch spezifische oder allgemeine Berechtigungen erteilt werden können.

Um Benutzer zu verwalten, führen Sie folgende Schritte aus:

  • Gehen Sie zu „Benutzer verwalten" im Menü „Verwaltung".
  • Das Programm zeigt ein Formular mit einer Benutzerliste an.
  • Klicken Sie auf die Bearbeiten-Schaltfläche des gewünschten Benutzers oder klicken Sie auf die Schaltfläche „Erstellen".
  • Es erscheint ein Formular mit folgenden Feldern:
    • Benutzername: Der Anmeldename des Benutzers.
    • Passwort: Das Passwort des Benutzers.
    • Autorisiert/Nicht autorisiert: Eine Einstellung zum Aktivieren oder Deaktivieren des Benutzerkontos.
    • E-Mail: Die E-Mail-Adresse des Benutzers.
    • Liste der zugehörigen Rollen: Um eine neue Rolle hinzuzufügen, müssen Benutzer in der Auswahlliste nach einer Rolle suchen und auf „Zuweisen" klicken.
    • Liste der zugehörigen Profile: Um ein neues Profil hinzuzufügen, müssen Benutzer in der Auswahlliste nach einem Profil suchen und auf „Zuweisen" klicken.

Profile verwalten

Um die Profile des Programms zu verwalten, müssen Benutzer folgende Schritte ausführen:

  • Gehen Sie zu „Benutzerprofile verwalten" im Menü „Verwaltung".
  • Das Programm zeigt eine Liste der Profile an.
  • Klicken Sie auf die Bearbeiten-Schaltfläche des gewünschten Profils oder klicken Sie auf „Erstellen".
  • Im Programm erscheint ein Formular mit folgenden Feldern:
    • Name: Der Name des Benutzerprofils.
    • Liste der Rollen (Berechtigungen): Um dem Profil eine Rolle hinzuzufügen, müssen Benutzer eine Rolle aus der Rollenliste auswählen und auf „Hinzufügen" klicken.

Benutzerprofile verwalten

  • Klicken Sie auf „Speichern" oder „Speichern und weiter", und das System speichert das erstellte oder geänderte Profil.
