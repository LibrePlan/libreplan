Etiketten sind Entitäten, die im Programm verwendet werden, um Aufgaben oder Elemente konzeptionell zu organisieren oder zu ordnen.

Etiketten werden nach Etikettentyp kategorisiert. Ein Etikett kann nur zu einem Etikettentyp gehören; Benutzer können jedoch viele ähnliche Etiketten erstellen, die zu verschiedenen Etikettentypen gehören.

Etikettentypen

Etikettentypen werden verwendet, um die Arten von Etiketten zu gruppieren, die Benutzer im Programm verwalten möchten. Hier sind einige Beispiele für mögliche Etikettentypen:

  • Auftraggeber: Benutzer können daran interessiert sein, Aufgaben, Aufträge oder Auftragsbestandteile in Bezug auf den Auftraggeber zu kennzeichnen, der sie anfordert.
  • Bereich: Benutzer können daran interessiert sein, Aufgaben, Aufträge oder Auftragsbestandteile in Bezug auf die Bereiche zu kennzeichnen, in denen sie durchgeführt werden.

Die Verwaltung von Etikettentypen erfolgt über die Menüoption „Verwaltung". Hier können Benutzer Etikettentypen bearbeiten, neue Etikettentypen erstellen und Etiketten zu Etikettentypen hinzufügen. Benutzer können über diese Option auf die Liste der Etiketten zugreifen.

Liste der Etikettentypen

Aus der Liste der Etikettentypen können Benutzer:

  • Einen neuen Etikettentyp erstellen.
  • Einen vorhandenen Etikettentyp bearbeiten.
  • Einen Etikettentyp mit allen seinen Etiketten löschen.

Das Bearbeiten und Erstellen von Etiketten verwenden dasselbe Formular. Über dieses Formular kann der Benutzer dem Etikettentyp einen Namen zuweisen, Etiketten erstellen oder löschen und die Änderungen speichern. Das Vorgehen ist wie folgt:

  • Wählen Sie ein zu bearbeitendes Etikett aus oder klicken Sie auf die Schaltfläche „Erstellen" für ein neues.
  • Das System zeigt ein Formular mit einem Texteingabefeld für den Namen und eine Liste von Texteingabefeldern mit vorhandenen und zugewiesenen Etiketten an.
  • Wenn Benutzer ein neues Etikett hinzufügen möchten, müssen sie auf die Schaltfläche „Neues Etikett" klicken.
  • Das System zeigt eine neue Zeile in der Liste mit einem leeren Textfeld an, das Benutzer bearbeiten müssen.
  • Benutzer geben einen Namen für das Etikett ein.
  • Das System fügt den Namen zur Liste hinzu.
  • Benutzer klicken auf „Speichern" oder „Speichern und weiter", um das Formular weiter zu bearbeiten.

Etiketten

Etiketten sind Entitäten, die zu einem Etikettentyp gehören. Diese Entitäten können Auftragsbestandteilen zugewiesen werden. Das Zuweisen eines Etiketts zu einem Auftragsbestandteil bedeutet, dass alle von diesem Element abstammenden Elemente das Etikett erben, zu dem sie gehören. Das Vorhandensein eines zugewiesenen Etiketts bedeutet, dass diese Entitäten gefiltert werden können, wo Suchen durchgeführt werden können:

  • Suche nach Aufgaben im Gantt-Diagramm.
  • Suche nach Auftragsbestandteilen in der Liste der Auftragsbestandteile.
  • Filter für Berichte.

Die Zuweisung von Etiketten zu Auftragsbestandteilen wird im Kapitel über Aufträge behandelt.
