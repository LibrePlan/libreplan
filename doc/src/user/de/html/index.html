LibrePlan: Benutzerdokumentation

Dieses Dokument enthält die notwendige Hilfe zur Verwendung von LibrePlan, der Open-Source-Webanwendung für die Projektplanung.

Das Dokument ist in drei Hauptabschnitte gegliedert:

Erstens, globale Ziele und Verhalten der Anwendung.

Zweitens, grundlegende Entitäten zum Verständnis der Mindestkonzepte für die Nutzung von LibrePlan.

Abschließend eine vollständige Prozessbeschreibung zur Erstellung von Aufträgen, Projekten, Projektplanung, Ressourcenzuweisung, Fortschrittserfassung und Ergebnisauswertung.
