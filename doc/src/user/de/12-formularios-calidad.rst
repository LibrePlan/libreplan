Verwaltung von Qualitätsformularen
===================================

Qualitätsformulare bestehen aus einer Liste von Fragen oder Aussagen, die angeben, welche Aufgaben oder Prozesse abgeschlossen sein sollten, bevor eine Aufgabe vom Unternehmen als abgeschlossen markiert werden kann. Diese Formulare enthalten folgende Felder:

*   **Name:** Der Name des Qualitätsformulars.
*   **Beschreibung:** Eine Beschreibung des Qualitätsformulars.
*   **Art des Qualitätsformulars:** Der Typ kann zwei Werte haben:

    *   **Prozentsatz:** Gibt an, dass die Fragen in einer logischen Reihenfolge beantwortet werden müssen und bejahende Antworten anzeigen, dass die Aufgabe voranschreitet. Beispielsweise könnte ein grundlegender Prozess für eine Aufgabe darauf hinweisen, dass die Aufgabe zu 15 % abgeschlossen ist. Benutzer müssen eine Frage beantworten, bevor sie zur nächsten übergehen.
    *   **Element:** Gibt an, dass die Fragen nicht in einer logischen Reihenfolge beantwortet werden müssen und in beliebiger Reihenfolge beantwortet werden können.

Benutzer müssen folgende Schritte ausführen, um Qualitätsformulare zu verwalten:

*   Öffnen Sie im Menü „Verwaltung" die Option „Qualitätsformulare".
*   Klicken Sie bei einem vorhandenen Formular auf „Bearbeiten" oder klicken Sie auf „Erstellen", um ein neues zu erstellen.
*   Das Programm zeigt ein Formular mit Feldern für Name, Beschreibung und Typ an.
*   Wählen Sie den Typ aus.
*   Das Programm zeigt die für jeden Typ zulässigen Felder an:

    *   **Prozentsatz:** Frage und Prozentsatz.
    *   **Element:** Frage.

*   Klicken Sie auf „Speichern" oder „Speichern und weiter".
