資源工時報告
############

.. contents::

目的
====

本報告提取任務清單以及資源在指定時間段內投入其中的工時。多個篩選器允許使用者精煉查詢，以僅取得所需資訊並排除無關資料。

輸入參數與篩選器
================

* **日期**。
    * *類型*：選填。
    * *兩個日期欄位*：
        * *開始日期：* 這是要納入工時報告的最早日期。早於*開始日期*的工時報告將被排除。若此參數留空，工時報告將不按*開始日期*篩選。
        * *結束日期：* 這是要納入工時報告的最晚日期。晚於*結束日期*的工時報告將被排除。若此參數留空，工時報告將不按*結束日期*篩選。

*   **依工人篩選：**
    *   *類型：* 選填。
    *   *運作方式：* 可以選擇一個或多個工人，以將工時報告限制為這些特定工人所追蹤的工時。若要新增工人作為篩選器，在選擇器中搜尋並點擊*新增*按鈕。若此篩選器留空，則不論工人為何，均會取得工時報告。

*   **依標籤篩選：**
    *   *類型：* 選填。
    *   *運作方式：* 可以透過在選擇器中搜尋並點擊*新增*按鈕，新增一個或多個標籤作為篩選器。這些標籤用於選擇要在計算投入工時時納入結果的任務。此篩選器可應用於工時表、任務、兩者或兩者皆不。

*   **依標準篩選：**
    *   *類型：* 選填。
    *   *運作方式：* 可以透過在選擇器中搜尋並點擊*新增*按鈕，選擇一個或多個標準。這些標準用於選擇至少滿足其中一個標準的資源。報告將顯示符合所選標準之一的資源所投入的所有工時。

輸出
====

標題
----

報告標題顯示為目前報告設定和應用的篩選器。

頁尾
----

頁尾列出報告生成的日期。

內文
----

報告內文由多組資訊組成。

*   第一層聚合是依資源。資源投入的所有工時顯示在標題下方。每個資源的識別方式如下：

    *   *工人：* 姓氏、名字。
    *   *機器：* 名稱。

    摘要行顯示該資源的總工時。

*   第二層分組是依*日期*。特定資源在同一日期的所有報告顯示在一起。

    摘要行顯示該資源在該日期的總工時。

*   最後一層列出工人當天的工時報告。每個工時報告行顯示的資訊為：

    *   *任務代碼：* 已追蹤工時所歸屬任務的代碼。
    *   *任務名稱：* 已追蹤工時所歸屬任務的名稱。
    *   *開始時間：* 選填。資源開始處理該任務的時間。
    *   *結束時間：* 選填。資源在指定日期完成該任務的時間。
    *   *文字欄位：* 選填。若工時報告行有文字欄位，則在此顯示填入的值。格式為：<文字欄位名稱>:<值>
    *   *標籤：* 這取決於工時報告模型的定義中是否有標籤欄位。若有多個標籤，則在同一欄顯示。格式為：<標籤類型名稱>:<標籤值>
