開放原始碼

LibrePlan 是一個免費的開放原始碼網路應用程式，用於專案規劃。原始碼公開提供，任何人均可使用、研究、修改及重新散布。不收取任何授權費用。

開發和維護這種規模的專案需要貢獻者持續付出努力。由於 LibrePlan 不收取軟體費用，本專案依賴自願貢獻——包括程式碼和財務捐款——才能持續運作。

登入畫面上的捐款按鈕

當您開啟 LibrePlan 時，您會注意到登入畫面上有一個 捐款 按鈕。此按鈕是對您在財務上支持本專案的直接邀請。捐款有助於資助：

  • 持續維護和錯誤修正
  • 新功能和改進
  • 基礎設施費用（伺服器、CI、託管）
  • 文件和翻譯工作

即使是小額的一次性捐款，對維持應用程式運作的志工和開發者而言也會產生真實的影響。如果 LibrePlan 對您或您的組織有所幫助，請考慮點擊按鈕並進行捐款。

適用於企業的訂閱版本

LibrePlan 的訂閱版本（託管或本地部署）可透過我們的姊妹公司 LibrePlan Enterprise 提供給企業使用，適合那些偏好有保障支援和 SLA 協議的受管服務的用戶。該版本的訂閱者在登入畫面上不會看到捐款按鈕，因為他們已透過訂閱費用為專案的永續發展做出貢獻。

如果您使用的是社群（自行託管）版本，捐款按鈕將會顯示，提醒您本專案依賴使用者的慷慨支持。
