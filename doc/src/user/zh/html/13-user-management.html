管理使用者

LibrePlan 的系統允許管理員管理使用者設定檔、授權和使用者。使用者被分配到使用者設定檔，這些設定檔可以有一系列預定義的角色，用於授予對程式功能的存取權限。角色是 LibrePlan 中定義的授權。角色的範例包括：

  • 管理： 必須分配給管理員的角色，以使他們能夠執行管理操作。
  • 網路服務讀取者： 使用者諮詢程式網路服務所需的角色。
  • 網路服務寫入者： 使用者透過程式的網路服務寫入資料所需的角色。

角色在系統中預先定義。使用者設定檔由一個或多個角色組成。使用者必須具有特定角色才能執行某些操作。

使用者可以被分配一個或多個設定檔，或者直接分配一個或多個角色，從而允許授予特定或通用授權。

要管理使用者，請遵循以下步驟：

  • 在「管理」選單中前往「管理使用者」。
  • 程式顯示帶有使用者清單的表單。
  • 點擊所需使用者的編輯按鈕，或點擊「建立」按鈕。
  • 將出現一個帶有以下欄位的表單：
    • 使用者名稱： 使用者的登入名稱。
    • 密碼： 使用者的密碼。
    • 已授權/未授權： 用於啟用或停用使用者帳戶的設定。
    • 電子郵件： 使用者的電子郵件地址。
    • 相關角色清單： 要新增新角色，使用者必須在選擇清單中搜尋角色，然後點擊「指定」。
    • 相關設定檔清單： 要新增新設定檔，使用者必須在選擇清單中搜尋設定檔，然後點擊「指定」。

管理使用者

  • 點擊「儲存」或「儲存並繼續」。

管理設定檔

要管理程式的設定檔，使用者必須遵循以下步驟：

  • 在「管理」選單中前往「管理使用者設定檔」。
  • 程式顯示設定檔清單。
  • 點擊所需設定檔的編輯按鈕，或點擊「建立」。
  • 程式中出現一個帶有以下欄位的表單：
    • 名稱： 使用者設定檔的名稱。
    • 角色（授權）清單： 要向設定檔新增角色，使用者必須從角色清單中選擇一個角色，然後點擊「新增」。

管理使用者設定檔

  • 點擊「儲存」或「儲存並繼續」，系統將儲存已建立或修改的設定檔。
