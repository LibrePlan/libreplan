LibrePlan: 使用者說明文件

本文件包含使用 LibrePlan 所需的說明，LibrePlan 是一款用於專案規劃的開源網頁應用程式。

文件分為三個主要部分：

第一部分，全域目標與整體行為。

第二部分，了解使用 LibrePlan 所需最低概念的基本實體。

最後，建立訂單、專案、專案規劃、資源分配、進度登錄與結果提取的完整流程說明。
