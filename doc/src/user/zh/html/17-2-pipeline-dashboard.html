流水線儀表板

流水線儀表板是 LibrePlan 的一個視角，以表格形式列出公司的所有專案。這個「流水線」表格有九個欄位，每個欄位對應一種專案狀態：

  • PRE-SALES（售前）
  • OFFERED（已提案）
  • OUTSOURCED（外包）
  • ACCEPTED（已接受）
  • STARTED（已開始）
  • ON HOLD（暫停）
  • FINISHED（已完成）
  • CANCELLED（已取消）
  • ARCHIVED（已封存）

專案根據其目前狀態，依名稱列在對應欄位中。

將滑鼠游標懸停在專案名稱上，可在工具提示彈出視窗中查看專案的開始日期、結束日期及目前進度。

狀態為 ARCHIVED 的專案可透過取消勾選「顯示封存欄位資料」核取方塊來隱藏。
