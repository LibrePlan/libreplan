工作報告
########

.. contents::

工作報告使人們能夠監控資源投入到其被分配的任務的工時。

程式允許使用者設定用於輸入投入工時的新表單，指定他們希望在這些表單中出現的欄位。這允許納入員工執行任務的報告，以及監控員工活動。

在使用者可以為資源新增條目之前，他們必須定義至少一種工作報告類型。此類型定義報告的結構，包括新增到其中的所有行。使用者可以在系統中根據需要建立任意數量的工作報告類型。

工作報告類型
============

工作報告由一系列對整個報告通用的欄位和一組具有每行定義的欄位的特定值的工作報告行組成。例如，資源和任務對所有報告通用。但是，可能有其他新欄位，例如「事件」，這在所有報告類型中都不是必需的。

使用者可以設定不同的工作報告類型，以便公司可以設計其報告以滿足其特定需求：

.. figure:: images/work-report-types.png
   :scale: 40

   工作報告類型

工作報告類型的管理允許使用者設定這些類型並新增新的文字欄位或選填標籤。在編輯工作報告類型的第一個索引標籤中，可以為必填屬性（無論它們適用於整個報告還是在行級別指定）設定類型，並新增新的選填欄位。

所有工作報告中必須出現的必填欄位如下：

*   **名稱和代碼：** 工作報告類型名稱及其代碼的識別欄位。
*   **日期：** 報告日期的欄位。
*   **資源：** 出現在報告或工作報告行上的員工或機器。
*   **專案元素：** 所執行工作被歸因到的專案元素的代碼。
*   **工時管理：** 決定要使用的工時歸因政策，可以是：

    *   **根據分配工時：** 工時根據分配的工時進行歸因。
    *   **根據開始和完成時間：** 工時根據開始和完成時間計算。
    *   **根據工時數和開始/完成範圍：** 允許差異，工時數優先。

使用者可以向報告新增新欄位：

*   **標籤類型：** 使用者可以請求系統在完成工作報告時顯示標籤。例如，客戶標籤類型，如果使用者希望在每份報告中輸入執行工作的客戶。
*   **自由欄位：** 可以在工作報告中自由輸入文字的欄位。

.. figure:: images/work-report-type.png
   :scale: 50

   建立具有個人化欄位的工作報告類型

使用者可以設定日期、資源和專案元素欄位出現在報告標題中，這意味著它們適用於整個報告，或者它們可以新增到每一行。

最後，可以使用「附加文字」和「標籤類型」欄位，分別在工作報告標題或每行中，向現有欄位新增新的附加文字欄位或標籤。使用者可以在「管理附加欄位和標籤」索引標籤中設定這些元素的輸入順序。

工作報告清單
============

一旦要納入系統的報告格式設定完成，使用者可以根據對應工作報告類型中定義的結構，在建立的表單中輸入詳細資訊。為此，使用者需要遵循以下步驟：

*   從工作報告類型清單中，點擊與所需報告相關聯的「新工作報告」按鈕。
*   程式然後根據為類型提供的設定顯示報告。見下圖。

.. figure:: images/work-report-type.png
   :scale: 50

   基於類型的工作報告結構

*   選擇報告所示的所有欄位：

    *   **資源：** 如果選擇了標題，資源只顯示一次。或者，對於報告的每一行，都需要選擇一個資源。
    *   **任務代碼：** 正在分配工作報告的任務的代碼。與其他欄位類似，如果欄位在標題中，則在報告行中輸入一次或根據需要輸入多次。
    *   **日期：** 報告或每行的日期，取決於標題或行是否已設定。
    *   **工時數：** 專案中的工作工時數。
    *   **開始和完成時間：** 工作的開始和完成時間，以計算確定的工作時數。此欄位僅在工時分配政策為「根據開始和完成時間」和「根據工時數和開始/完成範圍」的情況下出現。
    *   **工時類型：** 使使用者能夠選擇工時類型，例如「正常」、「加班」等。

*   點擊「儲存」或「儲存並繼續」。
