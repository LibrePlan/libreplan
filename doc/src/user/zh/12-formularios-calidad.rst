品質表單管理
============

品質表單由一份問題或陳述清單組成，指示在任務可以被公司標記為完成之前應完成的任務或流程。這些表單包含以下欄位：

*   **名稱：** 品質表單的名稱。
*   **描述：** 品質表單的描述。
*   **品質表單類型：** 類型可以有兩個值：

    *   **百分比：** 表示問題必須按邏輯順序回答，肯定的答案表示任務正在進行中。例如，任務的基本流程可能表明任務已完成 15%。使用者必須在進行下一個問題之前回答一個問題。
    *   **項目：** 表示問題不需要按邏輯順序回答，可以以任何順序回答。

使用者必須遵循以下步驟來管理品質表單：

*   從「管理」選單，存取「品質表單」選項。
*   在現有表單上點擊「編輯」，或點擊「建立」以建立新表單。
*   程式顯示帶有名稱、描述和類型欄位的表單。
*   選擇類型。
*   程式顯示每種類型允許的欄位：

    *   **百分比：** 問題和百分比。
    *   **項目：** 問題。

*   點擊「儲存」或「儲存並繼續」。
