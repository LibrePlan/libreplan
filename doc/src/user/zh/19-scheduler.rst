排程器
######

.. contents::

排程器設計用於動態排程作業，使用 *Spring Framework Quartz scheduler* 開發。

若要有效使用此排程器，必須先建立需要排程的作業（Quartz 作業）。然後，這些作業可以新增至資料庫，因為所有待排程的作業均儲存在資料庫中。

排程器啟動時，會從資料庫讀取待排程或待取消排程的作業，並依此排程或移除它們。之後，可以使用 ``作業排程`` 使用者介面動態新增、更新或移除作業。

.. NOTE::
   排程器在 LibrePlan 網路應用程式啟動時開始運行，在應用程式停止時停止。

.. NOTE::
   此排程器僅支援使用 ``cron 運算式`` 來排程作業。

排程器啟動時，用於排程或移除作業的判斷標準如下：

對於所有作業：

* 排程

  * 作業具有 *連接器*，且 *連接器* 已啟用，且作業允許被排程。
  * 作業沒有 *連接器*，且允許被排程。

* 移除

  * 作業具有 *連接器*，且 *連接器* 未啟用。
  * 作業具有 *連接器*，且 *連接器* 已啟用，但作業不允許被排程。
  * 作業沒有 *連接器*，且不允許被排程。

.. NOTE::
   若作業正在執行中，則無法重新排程或取消排程。

作業排程清單視圖
================

``作業排程清單`` 視圖允許使用者：

*   新增作業。
*   編輯現有作業。
*   移除作業。
*   手動啟動程序。

新增或編輯作業
==============

從 ``作業排程清單`` 視圖，點擊：

*   ``建立`` 以新增作業，或
*   ``編輯`` 以修改所選的作業。

兩個操作都將開啟建立/編輯 ``作業表單``。``表單`` 顯示以下屬性：

*   欄位：

    *   **作業群組：** 作業群組的名稱。
    *   **作業名稱：** 作業的名稱。
    *   **Cron 運算式：** 唯讀欄位，含 ``編輯`` 按鈕，可開啟 ``cron 運算式`` 輸入視窗。
    *   **作業類別名稱：** 用於選擇作業（現有作業）的 ``下拉式清單``。
    *   **連接器：** 用於選擇連接器的 ``下拉式清單``。此項非必填。
    *   **排程：** 核取方塊，用於指示是否排程此作業。

*   按鈕：

    *   **儲存：** 在資料庫和排程器中儲存或更新作業，並返回 ``作業排程清單視圖``。
    *   **儲存並繼續：** 與「儲存」相同，但不返回 ``作業排程清單視圖``。
    *   **取消：** 不儲存任何內容，並返回 ``作業排程清單視圖``。

*   以及關於 cron 運算式語法的提示區段。

Cron 運算式彈出視窗
-------------------

若要正確輸入 ``cron 運算式``，請使用 ``cron 運算式`` 彈出表單。在此表單中，您可以輸入所需的 ``cron 運算式``。另請參閱關於 ``cron 運算式`` 的提示。若輸入無效的 ``cron 運算式``，將立即收到通知。

移除作業
========

點擊 ``移除`` 按鈕，從資料庫和排程器中刪除作業。此操作的成功或失敗結果將予以顯示。

手動啟動作業
============

除了等待作業依排程執行外，您也可以點擊此按鈕直接啟動程序。之後，成功或失敗的資訊將顯示在 ``彈出視窗`` 中。
