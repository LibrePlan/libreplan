材料管理
========

使用者可以管理按類別組織的基本材料資料庫。

類別是容器，可以向其分配特定材料和其他類別。它們以層次樹狀結構儲存，因為材料可以屬於葉類別或中間類別。

要管理類別，使用者必須遵循以下步驟：

*   前往「管理 -> 材料」選項。
*   程式顯示類別樹。
*   在文字方塊中輸入類別名稱，然後點擊「新增」。
*   程式將類別新增到樹中。

要將類別插入到類別樹中，使用者必須先在樹中選擇父類別，然後點擊「新增」。

.. figure:: images/material.png
   :scale: 50

   材料管理畫面

要管理材料，使用者必須遵循以下步驟：

*   選擇要新增材料的類別，然後點擊「材料」右側的「新增」。
*   程式新增一個帶有輸入材料詳細資訊欄位的空行：

    *   **代碼：** 材料類型代碼（這可以是來自 ERP 的外部代碼）。
    *   **描述：** 材料的描述。
    *   **單位價格：** 每件材料的單位價格。
    *   **單位：** 用於衡量每個材料單位的單位。
    *   **類別：** 材料所屬的類別。
    *   **可用性：** 指示材料是否處於啟用狀態。

*   使用者填寫欄位，然後點擊「儲存」。

材料分配給專案元素在「專案」章節中說明。
