Dashboard della Pipeline
########################

.. contents::

Il dashboard della pipeline è una prospettiva di *LibrePlan* che elenca tutti i progetti aziendali in una tabella. Questa tabella "pipeline" ha nove colonne, ciascuna corrispondente a uno stato del progetto:

   *   PRE-VENDITA
   *   OFFERTO
   *   ESTERNALIZZATO
   *   ACCETTATO
   *   AVVIATO
   *   IN ATTESA
   *   TERMINATO
   *   ANNULLATO
   *   ARCHIVIATO

I progetti sono elencati per nome in ciascuna colonna in base al loro stato corrente.

La data di inizio, la data di fine e l'avanzamento corrente di un progetto possono essere visualizzati in un tooltip passando il cursore del mouse sopra il nome del progetto.

I progetti con stato *ARCHIVIATO* possono essere nascosti deselezionando la casella di controllo "Mostra dati colonna archiviata".
