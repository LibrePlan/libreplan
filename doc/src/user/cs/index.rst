##################################
LibrePlan: Uživatelská dokumentace
##################################

.. image:: images/logo.png
   :align: left

Tento dokument obsahuje nápovědu potřebnou pro používání LibrePlan, open-source webové aplikace pro plánování projektů.

Dokument je rozdělen do tří hlavních částí:

Za prvé, globální cíle a chování aplikace.

Za druhé, základní entity pro pochopení minimálních konceptů potřebných k používání LibrePlan.

Za třetí, úplný popis procesů pro vytváření zakázek, projektů, plánování projektů, přidělování zdrojů, zadávání pokroku a extrakci výsledků.

.. topic:: Prosím, podpořte LibrePlan
   :class: highlight
