Державне фінансування
======================

Цей проєкт наразі приймає пожертвування через організацію Open Collective <https://opencollective.com/LibrePlan>.

У межах глобального охоплення, для якого LibrePlan призначений щодо управління плануванням, був розроблений проєкт для вирішення деяких поширених проблем планування.

Цей проєкт частково фінансувався Xunta de Galicia, Ministerio de Industria, Turismo e Comercio та Європейським Союзом, Fondo Europeo de Desenvolvemento Rexional.
