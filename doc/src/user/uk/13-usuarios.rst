Управління користувачами
=========================

Система LibrePlan дозволяє адміністраторам управляти профілями користувачів, авторизаціями та користувачами. Користувачі призначаються до профілів користувачів, що можуть мати ряд попередньо визначених ролей, що надають доступ до функцій програми. Ролі — це визначені авторизації в LibrePlan. Приклади ролей:

*   **Адміністрування:** Роль, яку необхідно призначити адміністраторам, щоб вони могли виконувати адміністративні операції.
*   **Читач вебсервісів:** Роль, необхідна для того, щоб користувачі могли звертатися до вебсервісів програми.
*   **Записувач вебсервісів:** Роль, необхідна для того, щоб користувачі могли записувати дані через вебсервіси програми.

Ролі попередньо визначені в системі. Профіль користувача складається з однієї або кількох ролей. Для виконання певних операцій користувачі повинні мати конкретні ролі.

Користувачам можна призначати один або кілька профілів чи одну або кілька ролей безпосередньо, що дозволяє надавати конкретну або загальну авторизацію.

Для управління користувачами виконайте такі кроки:

*   Перейдіть до «Управління користувачами» в меню «Адміністрування».
*   Програма відображає форму зі списком користувачів.
*   Натисніть кнопку редагування для потрібного користувача або натисніть кнопку «Створити».
*   З'явиться форма з такими полями:

    *   **Ім'я користувача:** Логін користувача.
    *   **Пароль:** Пароль користувача.
    *   **Авторизований/Неавторизований:** Налаштування для ввімкнення або вимкнення облікового запису користувача.
    *   **Електронна пошта:** Адреса електронної пошти користувача.
    *   **Список пов'язаних ролей:** Щоб додати нову роль, користувачі повинні знайти роль у списку вибору та натиснути «Призначити».
    *   **Список пов'язаних профілів:** Щоб додати новий профіль, користувачі повинні знайти профіль у списку вибору та натиснути «Призначити».

Управління профілями
---------------------

Для управління профілями програми користувачі повинні виконати такі кроки:

*   Перейдіть до «Управління профілями користувачів» в меню «Адміністрування».
*   Програма відображає список профілів.
*   Натисніть кнопку редагування для потрібного профілю або натисніть «Створити».
*   У програмі з'явиться форма з такими полями:

    *   **Назва:** Назва профілю користувача.
    *   **Список ролей (авторизацій):** Щоб додати роль до профілю, користувачі повинні вибрати роль зі списку ролей та натиснути «Додати».

.. figure:: images/manage-user-profile.png
   :scale: 50

   Управління профілями користувачів

*   Натисніть «Зберегти» або «Зберегти і продовжити», і система збереже створений або змінений профіль.
