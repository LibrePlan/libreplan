Витрати
========

Управління витратами дозволяє користувачам оцінювати витрати на ресурси, використані в проєкті. Для управління витратами необхідно визначити такі сутності:

*   **Типи годин:** Вказують на типи годин, відпрацьованих ресурсом. Користувачі можуть визначати типи годин як для машин, так і для працівників. Приклади типів годин: «Додаткові години за ставкою 20 євро на годину». Для типів годин можна визначити такі поля:

    *   **Код:** Зовнішній код для типу годин.
    *   **Назва:** Назва типу годин. Наприклад, «Додаткові».
    *   **Стандартна ставка:** Базова стандартна ставка для типу годин.
    *   **Активація:** Вказує, чи активний тип годин.

*   **Категорії витрат:** Категорії витрат визначають витрати, пов'язані з різними типами годин протягом конкретних періодів (які можуть бути безстроковими). Наприклад, вартість додаткових годин для кваліфікованих працівників першого розряду в наступному році становить 24 євро на годину. Категорії витрат включають:

    *   **Назва:** Назва категорії витрат.
    *   **Активація:** Вказує, чи активна категорія.
    *   **Список типів годин:** Цей список визначає типи годин, включені до категорії витрат. Він визначає періоди та ставки для кожного типу годин. Наприклад, зі зміною ставок кожного року можна включати до цього списку новий рядок з конкретною погодинною ставкою для кожного типу годин (яка може відрізнятися від стандартної погодинної ставки для цього типу годин).

Управління типами годин
------------------------

Для реєстрації типів годин користувачі повинні виконати такі кроки:

*   Виберіть «Управління типами відпрацьованих годин» в меню «Адміністрування».
*   Програма відображає список наявних типів годин.

.. figure:: images/hour-type-list.png
   :scale: 35

   Список типів годин

*   Натисніть «Редагувати» або «Створити».
*   Програма відображає форму редагування типів годин.

.. figure:: images/hour-type-edit.png
   :scale: 50

   Редагування типів годин

*   Користувачі можуть вводити або змінювати:

    *   Назву типу годин.
    *   Код типу годин.
    *   Стандартну ставку.
    *   Активацію/деактивацію типу годин.

*   Натисніть «Зберегти» або «Зберегти і продовжити».

Категорії витрат
-----------------

Для реєстрації категорій витрат користувачі повинні виконати такі кроки:

*   Виберіть «Управління категоріями витрат» в меню «Адміністрування».
*   Програма відображає список наявних категорій.

.. figure:: images/category-cost-list.png
   :scale: 50

   Список категорій витрат

*   Натисніть кнопку «Редагувати» або «Створити».
*   Програма відображає форму редагування категорій витрат.

.. figure:: images/category-cost-edit.png
   :scale: 50

   Редагування категорій витрат

*   Користувачі вводять або змінюють:

    *   Назву категорії витрат.
    *   Активацію/деактивацію категорії витрат.
    *   Список типів годин, включених до категорії. Усі типи годин мають такі поля:

        *   **Тип годин:** Виберіть один з наявних типів годин у системі. Якщо жодного не існує, необхідно створити тип годин (цей процес пояснюється в попередньому підрозділі).
        *   **Дата початку та завершення:** Дати початку та завершення (остання є необов'язковою) для періоду, що застосовується до категорії витрат.
        *   **Погодинна ставка:** Погодинна ставка для цієї конкретної категорії.

*   Натисніть «Зберегти» або «Зберегти і продовжити».

Призначення категорій витрат ресурсам описано в розділі про ресурси. Перейдіть до розділу «Ресурси».
