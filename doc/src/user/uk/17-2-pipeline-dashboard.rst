Панель конвеєра
################

.. contents::

Панель конвеєра — це перспектива *LibrePlan*, що відображає всі проєкти компанії у вигляді таблиці. Ця таблиця «конвеєра» має дев'ять стовпців, кожен з яких відповідає статусу проєкту:

   *   ПЕРЕДПРОДАЖНИЙ ЕТАП
   *   ЗАПРОПОНОВАНИЙ
   *   НА АУТСОРСИНГУ
   *   ПРИЙНЯТИЙ
   *   РОЗПОЧАТИЙ
   *   ПРИЗУПИНЕНИЙ
   *   ЗАВЕРШЕНИЙ
   *   СКАСОВАНИЙ
   *   АРХІВОВАНИЙ

Проєкти перелічуються за назвою в кожному стовпці залежно від їхнього поточного статусу.

Дату початку, дату завершення та поточний прогрес проєкту можна переглянути у спливаючій підказці, навівши курсор миші на назву проєкту.

Проєкти зі статусом *АРХІВОВАНИЙ* можна приховати, знявши прапорець «Показати дані стовпця архівованих».
