Адміністрування матеріалів
===========================

Користувачі можуть управляти базовою базою даних матеріалів, організованих за категоріями.

Категорії — це контейнери, до яких можна призначати конкретні матеріали та інші категорії. Вони зберігаються в ієрархічній деревоподібній структурі, оскільки матеріали можуть належати до кінцевих або проміжних категорій.

Для управління категоріями користувачі повинні виконати такі кроки:

*   Перейдіть до параметра «Адміністрування -> Матеріали».
*   Програма відображає дерево категорій.
*   Введіть назву категорії в текстовому полі та натисніть «Додати».
*   Програма додає категорію до дерева.

Для вставки категорії в дерево категорій користувачі повинні спочатку вибрати батьківську категорію в дереві, а потім натиснути «Додати».

   :scale: 50

   Екран адміністрування матеріалів

Для управління матеріалами користувачі повинні виконати такі кроки:

*   Виберіть категорію, до якої потрібно додати матеріали, і натисніть «Додати» праворуч від «Матеріали».
*   Програма додає новий порожній рядок з полями для введення деталей про матеріал:

    *   **Код:** Код типу матеріалу (це може бути зовнішній код з ERP).
    *   **Опис:** Опис матеріалу.
    *   **Ціна за одиницю:** Ціна за одиницю кожного елемента матеріалу.
    *   **Одиниця:** Одиниця, що використовується для вимірювання кожної одиниці матеріалу.
    *   **Категорія:** Категорія, до якої належить матеріал.
    *   **Доступність:** Вказує, чи активний матеріал.

*   Користувачі заповнюють поля та натискають «Зберегти».

Призначення матеріалів елементам проєктів пояснюється в розділі «Проєкти».
