Contents

  • 标准管理

标准是程序中用于对资源和任务进行分类的元素。任务需要特定的标准，而资源必须符合这些标准。

以下是标准使用方式的示例：一个资源被分配了「焊接工」标准（意味着该资源符合「焊接工」类别），而一个任务需要「焊接工」标准才能完成。因此，当使用通用分配（而非特定分配）将资源分配给任务时，具有「焊接工」标准的员工将被纳入考量。有关不同分配类型的更多信息，请参阅资源分配章节。

程序允许对标准执行以下操作：

  • 标准管理
  • 为资源分配标准
  • 为任务分配标准
  • 根据标准筛选实体。可以按标准筛选任务和项目项目，以在程序中执行各种操作。

本节仅说明第一个功能：标准管理。两种分配类型将在后面说明：「资源管理」章节中的资源分配，以及「任务规划」章节中的筛选。

标准管理

可通过管理菜单访问标准管理：

一级菜单选项卡

管理标准的具体操作是 管理标准。此操作允许您列出系统中可用的标准。

标准列表

您可以通过点击 创建 按钮来访问创建/编辑标准表单。要编辑现有标准，请点击编辑图标。

编辑标准

如上图所示，标准编辑表单允许您执行以下操作：

  • 编辑标准的名称。
  • 指定是否可以同时分配多个值，或者所选标准类型只能有一个值。 例如，一个资源可以同时符合「焊接工」和「车床操作员」两个标准。
  • 指定标准类型：
    • 通用： 可用于机器和员工的标准。
    • 员工： 只能用于员工的标准。
    • 机器： 只能用于机器的标准。
  • 指出标准是否为层次结构。 有时，标准需要以层次结构方式处理。例如，将标准分配给一个元素不会自动将其分配给由其派生的元素。层次结构标准的一个清晰示例是「位置」。例如，被指定为「加利西亚」位置的人也会属于「西班牙」。
  • 指出标准是否已授权。 这是用户停用标准的方式。一旦标准已创建并用于历史数据，就无法更改。取而代之，可以将其停用以防止其出现在选择列表中。
  • 描述标准。
  • 添加新值。 表单的第二部分有一个带有 新标准 按钮的文字输入字段。
  • 编辑现有标准值的名称。
  • 在当前标准值列表中上下移动标准值。
  • 从列表中移除标准值。

标准管理表单遵循介绍中描述的表单行为，提供三种操作：保存、保存并关闭 和 关闭。
