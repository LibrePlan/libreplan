管理用户

LibrePlan 的系统允许管理员管理用户配置文件、授权和用户。用户被分配到用户配置文件，用户配置文件可以具有一系列预定义角色，这些角色授予对程序功能的访问权限。角色是 LibrePlan 中定义的授权。角色示例包括：

  • 管理员： 必须分配给管理员的角色，使其能够执行管理操作。
  • Web 服务读取者： 用户查询程序 Web 服务所需的角色。
  • Web 服务写入者： 用户通过程序 Web 服务写入数据所需的角色。

角色在系统中预先定义。用户配置文件由一个或多个角色组成。用户必须具有特定角色才能执行某些操作。

用户可以被分配一个或多个配置文件，或直接分配一个或多个角色，从而授予特定或通用的授权。

要管理用户，请按照以下步骤操作：

  • 在「管理」菜单中前往「管理用户」。
  • 程序显示一个包含用户列表的表单。
  • 点击所需用户的编辑按钮，或点击「创建」按钮。
  • 将出现一个包含以下字段的表单：
    • 用户名： 用户的登录名。
    • 密码： 用户的密码。
    • 授权/未授权： 启用或禁用用户账户的设置。
    • 电子邮件： 用户的电子邮件地址。
    • 关联角色列表： 要添加新角色，用户必须在选择列表中搜索角色并点击「分配」。
    • 关联配置文件列表： 要添加新配置文件，用户必须在选择列表中搜索配置文件并点击「分配」。

管理用户

  • 点击「保存」或「保存并继续」。

管理配置文件

要管理程序的配置文件，用户必须按照以下步骤操作：

  • 在「管理」菜单中前往「管理用户配置文件」。
  • 程序显示配置文件列表。
  • 点击所需配置文件的编辑按钮，或点击「创建」。
  • 程序中将出现一个包含以下字段的表单：
    • 名称： 用户配置文件的名称。
    • 角色列表（授权）： 要将角色添加到配置文件，用户必须从角色列表中选择一个角色并点击「添加」。

管理用户配置文件

  • 点击「保存」或「保存并继续」，系统将存储已创建或修改的配置文件。
