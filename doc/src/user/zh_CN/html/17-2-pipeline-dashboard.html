流水线仪表板

流水线仪表板是 LibrePlan 的一个视角，以表格形式列出公司的所有项目。这个「流水线」表格有九个列，每列对应一种项目状态：

  • PRE-SALES（售前）
  • OFFERED（已提案）
  • OUTSOURCED（外包）
  • ACCEPTED（已接受）
  • STARTED（已开始）
  • ON HOLD（暂停）
  • FINISHED（已完成）
  • CANCELLED（已取消）
  • ARCHIVED（已归档）

项目根据其当前状态，按名称列在对应列中。

将鼠标光标悬停在项目名称上，可在工具提示弹出窗口中查看项目的开始日期、结束日期及当前进度。

状态为 ARCHIVED 的项目可通过取消勾选「显示归档列数据」复选框来隐藏。
