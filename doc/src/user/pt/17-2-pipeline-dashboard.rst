Painel de Pipeline
##################

.. contents::

O painel de pipeline é uma perspectiva do *LibrePlan* que lista todos os projectos da empresa numa tabela. Esta tabela de "pipeline" tem nove colunas, cada uma correspondente a um estado de projecto:

   *   PRE-SALES
   *   OFFERED
   *   OUTSOURCED
   *   ACCEPTED
   *   STARTED
   *   ON HOLD
   *   FINISHED
   *   CANCELLED
   *   ARCHIVED

Os projectos são listados por nome em cada coluna com base no seu estado actual.

A data de início, a data de fim e o progresso actual de um projecto podem ser visualizados numa janela de dica ao passar o cursor do rato sobre o nome do projecto.

Os projectos com o estado *ARCHIVED* podem ser ocultados desmarcando a caixa de verificação "Mostrar dados da coluna arquivada".
