Gerir Utilizadores

O sistema do LibrePlan permite aos administradores gerir perfis de utilizadores, autorizações e utilizadores. Os utilizadores são atribuídos a perfis de utilizadores, que podem ter uma série de funções predefinidas que concedem acesso às funcionalidades do programa. As funções são autorizações definidas no LibrePlan. Exemplos de funções incluem:

  • Administração: Uma função que deve ser atribuída a administradores para lhes permitir efectuar operações administrativas.
  • Leitor de Serviços Web: Uma função necessária para que os utilizadores consultem os serviços web do programa.
  • Escritor de Serviços Web: Uma função necessária para que os utilizadores escrevam dados através dos serviços web do programa.

As funções estão predefinidas no sistema. Um perfil de utilizador consiste em uma ou mais funções. Os utilizadores devem ter funções específicas para efectuar determinadas operações.

Os utilizadores podem ser atribuídos a um ou mais perfis, ou a uma ou mais funções directamente, permitindo que sejam concedidas autorizações específicas ou genéricas.

Para gerir utilizadores, siga estes passos:

  • Vá a "Gerir utilizadores" no menu "Administração".
  • O programa apresenta um formulário com uma lista de utilizadores.
  • Clique no botão de edição para o utilizador desejado ou clique no botão "Criar".
  • Aparecerá um formulário com os seguintes campos:
    • Nome de Utilizador: O nome de início de sessão do utilizador.
    • Palavra-passe: A palavra-passe do utilizador.
    • Autorizado/Não autorizado: Uma definição para activar ou desactivar a conta do utilizador.
    • E-mail: O endereço de e-mail do utilizador.
    • Lista de Funções Associadas: Para adicionar uma nova função, os utilizadores devem pesquisar uma função na lista de selecção e clicar em "Atribuir".
    • Lista de Perfis Associados: Para adicionar um novo perfil, os utilizadores devem pesquisar um perfil na lista de selecção e clicar em "Atribuir".

  • Clique em "Guardar" ou "Guardar e continuar".

Gestão de Perfis

Para gerir os perfis do programa, os utilizadores devem seguir estes passos:

  • Vá a "Gerir perfis de utilizadores" no menu "Administração".
  • O programa apresenta uma lista de perfis.
  • Clique no botão de edição para o perfil desejado ou clique em "Criar".
  • No programa aparece um formulário com os seguintes campos:
    • Nome: O nome do perfil de utilizador.
    • Lista de Funções (Autorizações): Para adicionar uma função ao perfil, os utilizadores devem seleccionar uma função da lista de funções e clicar em "Adicionar".

Gestão de Perfis de Utilizadores

  • Clique em "Guardar" ou "Guardar e continuar", e o sistema armazenará o perfil criado ou modificado.
