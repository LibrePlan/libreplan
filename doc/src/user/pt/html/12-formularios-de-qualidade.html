Administração de Formulários de Qualidade

Os formulários de qualidade consistem numa lista de questões ou afirmações que indicam as tarefas ou processos que devem ter sido concluídos antes de uma tarefa poder ser marcada como completa pela empresa. Estes formulários incluem os seguintes campos:

  • Nome: O nome do formulário de qualidade.
  • Descrição: Uma descrição do formulário de qualidade.
  • Tipo de Formulário de Qualidade: O tipo pode ter dois valores:
    • Percentagem: Indica que as questões devem ser respondidas numa ordem lógica, e as respostas afirmativas indicam que a tarefa está a progredir. Por exemplo, um processo básico para uma tarefa pode sugerir que a tarefa está 15% concluída. Os utilizadores devem responder a uma questão antes de prosseguir para a seguinte.
    • Item: Indica que as questões não precisam de ser respondidas numa ordem lógica e podem ser respondidas em qualquer sequência.

Os utilizadores devem seguir estes passos para gerir formulários de qualidade:

  • No menu "Administração", aceda à opção "Formulários de qualidade".
  • Clique em "Editar" num formulário existente ou clique em "Criar" para criar um novo.
  • O programa apresenta um formulário com campos para o nome, descrição e tipo.
  • Seleccione o tipo.
  • O programa apresenta os campos que são permitidos para cada tipo:
    • Percentagem: Questão e percentagem.
    • Item: Questão.
  • Clique em "Guardar" ou "Guardar e continuar".
