As etiquetas são entidades utilizadas no programa para organizar conceptualmente tarefas ou elementos de encomenda.

As etiquetas são categorizadas de acordo com o tipo de etiqueta. Uma etiqueta só pode pertencer a um tipo de etiqueta; no entanto, os utilizadores podem criar muitas etiquetas semelhantes pertencentes a diferentes tipos de etiquetas.

Tipos de Etiquetas

Os tipos de etiquetas são utilizados para agrupar os tipos de etiquetas que os utilizadores pretendem gerir no programa. Eis alguns exemplos de possíveis tipos de etiquetas:

  • Cliente: Os utilizadores podem estar interessados em etiquetar tarefas, encomendas ou elementos de encomenda em relação ao cliente que as solicita.
  • Área: Os utilizadores podem estar interessados em etiquetar tarefas, encomendas ou elementos de encomenda em relação às áreas em que são executadas.

A administração de tipos de etiquetas é gerida a partir da opção de menu "Administração". É aqui que os utilizadores podem editar tipos de etiquetas, criar novos tipos de etiquetas e adicionar etiquetas a tipos de etiquetas. Os utilizadores podem aceder à lista de etiquetas a partir desta opção.

Lista de Tipos de Etiquetas

A partir da lista de tipos de etiquetas, os utilizadores podem:

  • Criar um novo tipo de etiqueta.
  • Editar um tipo de etiqueta existente.
  • Eliminar um tipo de etiqueta com todas as suas etiquetas.

A edição e criação de etiquetas partilham o mesmo formulário. A partir deste formulário, o utilizador pode atribuir um nome ao tipo de etiqueta, criar ou eliminar etiquetas e guardar as alterações. O procedimento é o seguinte:

  • Seleccione uma etiqueta para editar ou clique no botão de criação para uma nova.
  • O sistema apresenta um formulário com uma entrada de texto para o nome e uma lista de entradas de texto com etiquetas existentes e atribuídas.
  • Se os utilizadores desejarem adicionar uma nova etiqueta, devem clicar no botão "Nova etiqueta".
  • O sistema apresenta uma nova linha na lista com uma caixa de texto vazia que os utilizadores devem editar.
  • Os utilizadores introduzem um nome para a etiqueta.
  • O sistema adiciona o nome à lista.
  • Os utilizadores clicam em "Guardar" ou "Guardar e continuar" para continuar a editar o formulário.

Etiquetas

As etiquetas são entidades que pertencem a um tipo de etiqueta. Estas entidades podem ser atribuídas a elementos de encomenda. Atribuir uma etiqueta a um elemento de encomenda significa que todos os elementos descendentes deste elemento herdarão a etiqueta à qual pertencem. Ter uma etiqueta atribuída significa que estas entidades podem ser filtradas onde podem ser efectuadas pesquisas:

  • Pesquisa de tarefas no diagrama de Gantt.
  • Pesquisa de elementos de encomenda na lista de elementos de encomenda.
  • Filtros para relatórios.

A atribuição de etiquetas a elementos de encomenda é abordada no capítulo sobre encomendas.
