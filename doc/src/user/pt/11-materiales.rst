Administração de Materiais
============================

Os utilizadores podem gerir uma base de dados básica de materiais, organizada por categorias.

As categorias são contentores aos quais podem ser atribuídos materiais específicos e outras categorias. São armazenadas numa estrutura hierárquica em árvore, pois os materiais podem pertencer a categorias folha ou categorias intermediárias.

Para gerir categorias, os utilizadores devem seguir estes passos:

*   Vá à opção "Administração -> Materiais".
*   O programa apresenta uma árvore de categorias.
*   Introduza um nome de categoria na caixa de texto e clique em "Adicionar".
*   O programa adiciona a categoria à árvore.

Para inserir uma categoria na árvore de categorias, os utilizadores devem primeiro seleccionar a categoria pai na árvore e depois clicar em "Adicionar".

.. figure:: images/material.png
   :scale: 50

   Ecrã de Administração de Materiais

Para gerir materiais, os utilizadores devem seguir estes passos:

*   Seleccione a categoria à qual os materiais devem ser adicionados e clique em "Adicionar" à direita de "Materiais".
*   O programa adiciona uma nova linha vazia com campos para introduzir detalhes sobre o material:

    *   **Código:** Código do tipo de material (pode ser o código externo de um ERP).
    *   **Descrição:** Descrição do material.
    *   **Preço por Unidade:** Preço unitário de cada peça de material.
    *   **Unidade:** Unidade utilizada para medir cada unidade de material.
    *   **Categoria:** Categoria à qual o material pertence.
    *   **Disponibilidade:** Indica se o material está activo ou não.

*   Os utilizadores preenchem os campos e clicam em "Guardar".

A atribuição de materiais a elementos de projeto é explicada no capítulo sobre "Projetos".
