Relatório do Total de Horas Trabalhadas por Recurso num Mês
############################################################

.. contents::

Finalidade
==========

Este relatório fornece o número total de horas trabalhadas por cada recurso num determinado mês. Estas informações podem ser úteis para determinar as horas extraordinárias dos trabalhadores ou, dependendo da organização, o número de horas pelas quais cada recurso deve ser remunerado.

A aplicação regista relatórios de trabalho tanto para trabalhadores como para máquinas. Para as máquinas, o relatório totaliza o número de horas em que estiveram em funcionamento durante o mês.

Parâmetros de Entrada e Filtros
================================

Para gerar este relatório, os utilizadores devem especificar o ano e o mês para os quais pretendem obter o número total de horas trabalhadas por cada recurso.

Resultado
=========

O formato de saída é o seguinte:

Cabeçalho
---------

O cabeçalho do relatório apresenta:

   *   O *ano* ao qual os dados do relatório dizem respeito.
   *   O *mês* ao qual os dados do relatório dizem respeito.

Rodapé
------

O rodapé apresenta a data em que o relatório foi gerado.

Corpo
-----

A secção de dados do relatório é constituída por uma única tabela com duas colunas:

   *   Uma coluna com o cabeçalho **Nome** para o nome do recurso.
   *   Uma coluna com o cabeçalho **Horas** com o número total de horas trabalhadas pelo recurso nessa linha.

Existe uma linha final que totaliza o número de horas trabalhadas por todos os recursos durante o *mês* e *ano* especificados.
