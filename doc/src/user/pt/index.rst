.. image:: images/logo.png

Este documento contém a ajuda necessária para utilizar o LibrePlan, a aplicação web de código aberto para o planeamento de projectos.

O documento está dividido em três secções principais:

Em primeiro lugar, os objectivos globais e o comportamento geral da aplicação.

Em segundo lugar, as entidades básicas para compreender os conceitos mínimos necessários para utilizar o LibrePlan.

Por fim, a descrição completa dos processos de criação de encomendas, projectos, planeamento de projectos, atribuição de recursos, registo de avanços e extracção de resultados.
