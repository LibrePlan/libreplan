Administração de Materiais

Os usuários podem gerenciar um banco de dados básico de materiais, organizado por categorias.

As categorias são contêineres aos quais podem ser atribuídos materiais específicos e outras categorias. São armazenadas em uma estrutura hierárquica em árvore, pois os materiais podem pertencer a categorias folha ou categorias intermediárias.

Para gerenciar categorias, os usuários devem seguir estes passos:

  • Vá à opção "Administração -> Materiais".
  • O programa exibe uma árvore de categorias.
  • Insira um nome de categoria na caixa de texto e clique em "Adicionar".
  • O programa adiciona a categoria à árvore.

Para inserir uma categoria na árvore de categorias, os usuários devem primeiro selecionar a categoria pai na árvore e depois clicar em "Adicionar".

Tela de Administração de Materiais

Para gerenciar materiais, os usuários devem seguir estes passos:

  • Selecione a categoria à qual os materiais devem ser adicionados e clique em "Adicionar" à direita de "Materiais".
  • O programa adiciona uma nova linha vazia com campos para inserir detalhes sobre o material:
    • Código: Código do tipo de material (pode ser o código externo de um ERP).
    • Descrição: Descrição do material.
    • Preço por Unidade: Preço unitário de cada peça de material.
    • Unidade: Unidade utilizada para medir cada unidade de material.
    • Categoria: Categoria à qual o material pertence.
    • Disponibilidade: Indica se o material está ativo ou não.
  • Os usuários preenchem os campos e clicam em "Salvar".

A atribuição de materiais a elementos de projeto é explicada no capítulo sobre "Projetos".
