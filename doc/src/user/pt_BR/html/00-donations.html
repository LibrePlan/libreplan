open-source

O LibrePlan é uma aplicação web gratuita e de código aberto para planejamento de projetos. O código-fonte está disponível publicamente e a aplicação pode ser utilizada, estudada, modificada e redistribuída por qualquer pessoa. Não existe nenhuma taxa de licença.

Desenvolver e manter um projeto desta dimensão requer um esforço sustentado dos seus colaboradores. Como o LibrePlan não cobra pelo seu software, o projeto depende de contribuições voluntárias — tanto na forma de código como de doações financeiras — para continuar.

O Botão de Doação na Tela de Login

Ao abrir o LibrePlan, você notará um botão Doar na tela de login. Este botão é um convite direto para apoiar financeiramente o projeto. As doações ajudam a financiar:

  • Manutenção contínua e correção de bugs
  • Novos recursos e melhorias
  • Custos de infraestrutura (servidores, CI, hospedagem)
  • Trabalho de documentação e tradução

Mesmo uma pequena contribuição pontual faz uma diferença real para os voluntários e desenvolvedores que mantêm a aplicação funcionando. Se o LibrePlan é útil para você ou para a sua organização, considere clicar no botão e fazer uma doação.

Versão por Assinatura para Empresas

Está disponível uma edição baseada em assinatura (hospedada ou instalada localmente) do LibrePlan para empresas através da nossa empresa parceira LibrePlan Enterprise, para aqueles que preferem um serviço gerenciado com suporte garantido e acordos de SLA. Os assinantes dessa edição não verão o botão de doação na tela de login, pois já contribuem para a sustentabilidade do projeto através da sua taxa de assinatura.

Se você estiver usando a edição comunitária (auto-hospedada), o botão de doação será visível como um lembrete de que o projeto depende da generosidade dos seus usuários.
