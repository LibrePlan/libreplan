Painel de Pipeline

O painel de pipeline é uma perspectiva do LibrePlan que lista todos os projetos da empresa em uma tabela. Esta tabela de "pipeline" tem nove colunas, cada uma correspondendo a um estado de projeto:

  • PRE-SALES
  • OFFERED
  • OUTSOURCED
  • ACCEPTED
  • STARTED
  • ON HOLD
  • FINISHED
  • CANCELLED
  • ARCHIVED

Os projetos são listados por nome em cada coluna com base no seu estado atual.

A data de início, a data de fim e o progresso atual de um projeto podem ser visualizados em uma janela de dica ao passar o cursor do mouse sobre o nome do projeto.

Os projetos com o estado ARCHIVED podem ser ocultados desmarcando a caixa de seleção "Mostrar dados da coluna arquivada".
