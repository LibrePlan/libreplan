código abierto
==============

LibrePlan es una aplicación web libre y de código abierto para la planificación de
proyectos. El código fuente está disponible públicamente y la aplicación puede ser usada,
estudiada, modificada y redistribuida por cualquier persona. No hay ningún coste de
licencia.

Desarrollar y mantener un proyecto de esta envergadura requiere un esfuerzo sostenido
de sus colaboradores. Dado que LibrePlan no cobra por su software, el proyecto depende
de contribuciones voluntarias — tanto en forma de código como de donaciones económicas —
para seguir adelante.

El botón de donación en el inicio de sesión
===========================================

Cuando abra LibrePlan, verá un botón **Donar** en la pantalla de inicio de sesión. Este
botón es una invitación directa a apoyar el proyecto económicamente. Las donaciones
ayudan a financiar:

-  Mantenimiento continuo y corrección de errores
-  Nuevas funcionalidades y mejoras
-  Costes de infraestructura (servidores, integración continua, alojamiento)
-  Trabajo de documentación y traducción

Incluso una pequeña contribución puntual marca una diferencia real para los voluntarios
y desarrolladores que mantienen la aplicación en funcionamiento. Si LibrePlan le resulta
útil a usted o a su organización, por favor considere hacer clic en el botón y realizar
una donación.

Versión de suscripción para empresas
=====================================

Existe una edición basada en suscripción (alojada o en las instalaciones propias) de
LibrePlan para empresas a través de nuestra empresa hermana `LibrePlan Enterprise
<https://libreplan-enterprise.com>`__, para quienes prefieran un servicio gestionado con
soporte garantizado y acuerdos de nivel de servicio. Los suscriptores de esa edición no
verán el botón de donación en la pantalla de inicio de sesión, ya que ya contribuyen a la
sostenibilidad del proyecto a través de su cuota de suscripción.

Si está utilizando la edición comunitaria (autoalojada), el botón de donación será visible
como recordatorio de que el proyecto depende de la generosidad de sus usuarios.
