Pipeline-dashboard
##################

.. contents::

Het pipeline-dashboard is een *LibrePlan*-perspectief dat alle bedrijfsprojecten in een tabel weergeeft. Deze "pipeline"-tabel heeft negen kolommen, elk overeenkomend met een projectstatus:

   *   PRE-SALES
   *   OFFERED
   *   OUTSOURCED
   *   ACCEPTED
   *   STARTED
   *   ON HOLD
   *   FINISHED
   *   CANCELLED
   *   ARCHIVED

Projecten worden op naam weergegeven in elke kolom op basis van hun huidige status.

De startdatum, einddatum en huidige voortgang van een project kunnen worden bekeken in een tooltip-pop-up door de muiscursor over de naam van een project te bewegen.

Projecten met de status *ARCHIVED* kunnen worden verborgen door het selectievakje "Kolom gearchiveerde gegevens weergeven" uit te vinken.
