Publieke financiering
=====================

Dit project accepteert momenteel donaties via de Open Collective-organisatie <https://opencollective.com/LibrePlan>.

Binnen het globale toepassingsgebied waarvoor LibrePlan is ontworpen met betrekking tot planningsbeheer, werd een project ontwikkeld om een aantal veelvoorkomende planningsproblemen op te lossen.

Dit project werd gedeeltelijk gefinancierd door de Xunta de Galicia, het Ministerio de Industria, Turismo e Comercio en de Europese Unie, Fondo Europeo de Desenvolvemento Rexional.
