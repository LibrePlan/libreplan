Rapporten

LibrePlan integreert met JasperReports voor rapportbeheer, waardoor gebruikers verschillende rapporten kunnen genereren voor het analyseren van de beschikbare gegevens in het programma.

De beschikbare rapporten zijn:

  • Rapport gewerkte uren per resource
  • Rapport totaal gewerkte uren per resource per maand
  • Rapport werk en voortgang per project
