LibrePlan: مستندات کاربری

این سند شامل راهنمای لازم برای استفاده از LibrePlan، برنامه وب متن‌باز برای برنامه‌ریزی پروژه است.

این سند به سه بخش اصلی تقسیم شده است:

اول، اهداف کلی و رفتار عمومی برنامه.

دوم، موجودیت‌های پایه برای درک حداقل مفاهیم لازم جهت استفاده از LibrePlan.

در پایان، توضیح کامل فرایندهای ایجاد سفارش‌ها، پروژه‌ها، برنامه‌ریزی پروژه، تخصیص منابع، ثبت پیشرفت و استخراج نتایج.

لطفاً از LibrePlan حمایت کنید

LibrePlan بدون حمایت شما نمی‌تواند وجود داشته باشد. لطفاً صفحه اهدا را بخوانید و در نظر بگیرید که به ما در ادامه توسعه کمک کنید.
