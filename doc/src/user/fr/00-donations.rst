open-source
===========

LibrePlan est une application web libre et open-source de planification de projets. Le
code source est accessible publiquement et l'application peut être utilisée, étudiée,
modifiée et redistribuée par quiconque. Il n'y a aucun frais de licence.

Développer et maintenir un projet de cette envergure nécessite un effort soutenu de la
part de ses contributeurs. Comme LibrePlan ne fait pas payer son logiciel, le projet
dépend de contributions volontaires — tant sous forme de code que de dons financiers —
pour continuer à exister.

Le bouton de don sur l'écran de connexion
==========================================

Lorsque vous ouvrez LibrePlan, vous remarquerez un bouton **Faire un don** sur l'écran
de connexion. Ce bouton est une invitation directe à soutenir financièrement le projet.
Les dons permettent de financer :

-  La maintenance continue et les corrections de bogues
-  Les nouvelles fonctionnalités et améliorations
-  Les coûts d'infrastructure (serveurs, intégration continue, hébergement)
-  Le travail de documentation et de traduction

Même une petite contribution ponctuelle fait une vraie différence pour les bénévoles et
les développeurs qui maintiennent l'application en fonctionnement. Si LibrePlan vous est
utile, à vous ou à votre organisation, veuillez envisager de cliquer sur le bouton et
d'effectuer un don.

Version par abonnement pour les entreprises
===========================================

Une édition par abonnement (hébergée ou sur site) de LibrePlan est disponible pour les
entreprises via notre société partenaire `LibrePlan Enterprise
<https://libreplan-enterprise.com>`__, pour celles qui préfèrent un service géré avec
support garanti et accords de niveau de service. Les abonnés à cette édition ne verront
pas le bouton de don sur l'écran de connexion, car ils contribuent déjà à la pérennité
du projet par leur abonnement.

Si vous utilisez l'édition communautaire (auto-hébergée), le bouton de don sera visible
en rappel que le projet repose sur la générosité de ses utilisateurs.
