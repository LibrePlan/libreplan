Les etiquetes són entitats que s'utilitzen al programa per organitzar conceptualment les tasques o els elements de comanda.

Les etiquetes es classifiquen d'acord amb el tipus d'etiqueta. Una etiqueta només pot pertànyer a un tipus d'etiqueta; no obstant això, els usuaris poden crear moltes etiquetes similars pertanyents a tipus d'etiquetes diferents.

Tipus d'Etiquetes

Els tipus d'etiquetes s'utilitzen per agrupar els tipus d'etiquetes que els usuaris volen gestionar al programa. Aquí hi ha alguns exemples de possibles tipus d'etiquetes:

  • Client: Els usuaris poden estar interessats a etiquetar tasques, comandes o elements de comanda en relació amb el client que les sol·licita.
  • Àrea: Els usuaris poden estar interessats a etiquetar tasques, comandes o elements de comanda en relació amb les àrees en les quals es duen a terme.

L'administració dels tipus d'etiquetes es gestiona des de l'opció de menú "Administració". Aquí és on els usuaris poden editar tipus d'etiquetes, crear nous tipus d'etiquetes i afegir etiquetes als tipus d'etiquetes. Els usuaris poden accedir a la llista d'etiquetes des d'aquesta opció.

Llista de Tipus d'Etiquetes

Des de la llista de tipus d'etiquetes, els usuaris poden:

  • Crear un nou tipus d'etiqueta.
  • Editar un tipus d'etiqueta existent.
  • Eliminar un tipus d'etiqueta amb totes les seves etiquetes.

L'edició i la creació d'etiquetes comparteixen el mateix formulari. Des d'aquest formulari, l'usuari pot assignar un nom al tipus d'etiqueta, crear o eliminar etiquetes i emmagatzemar els canvis. El procediment és el següent:

  • Seleccioneu una etiqueta per editar o feu clic al botó de creació per a una de nova.
  • El sistema mostra un formulari amb una entrada de text per al nom i una llista d'entrades de text amb les etiquetes existents i assignades.
  • Si els usuaris volen afegir una nova etiqueta, han de fer clic al botó "Nova etiqueta."
  • El sistema mostra una nova fila a la llista amb un quadre de text buit que els usuaris han d'editar.
  • Els usuaris introdueixen un nom per a l'etiqueta.
  • El sistema afegeix el nom a la llista.
  • Els usuaris fan clic a "Desar" o "Desar i continuar" per continuar editant el formulari.

Edició de Tipus d'Etiquetes

Etiquetes

Les etiquetes són entitats que pertanyen a un tipus d'etiqueta. Aquestes entitats es poden assignar a elements de comanda. Assignar una etiqueta a un element de comanda significa que tots els elements descendents d'aquest element heretaran l'etiqueta a la qual pertanyen. Tenir una etiqueta assignada significa que aquestes entitats es poden filtrar on es poden dur a terme cerques:

  • Cerca de tasques al diagrama de Gantt.
  • Cerca d'elements de comanda a la llista d'elements de comanda.
  • Filtres per als informes.

L'assignació d'etiquetes als elements de comanda s'explica al capítol de comandes.
